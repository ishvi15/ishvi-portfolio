# Ishvi Jain — Portfolio

Personal portfolio website showcasing my background, experience, and projects in Data Science and Machine Learning.

Built with Next.js, React, and Tailwind CSS.

## Live Site

[View Portfolio](https://ishvi-portfolio.vercel.app)

## Sections

- About
- Experience
- Skills
- Projects
- Education
- Contact

## Running Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- Next.js
- React
- Tailwind CSS
